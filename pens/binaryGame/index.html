The Binary Code Game

A Javascript representation of the game as seen in Harvard's CS50 course.

First get a number (click the get a number button), then click the circles in order to sum the values and match the number you got, without going over it. Once you get it right, what you actually see is a binary representation of that number. Clicking the same value twice negates the purpose of the game.
